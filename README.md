# 🎮 Is it SAFE BomB

![Unity](https://img.shields.io/badge/Unity-2021.3+-black.svg)
![Platform](https://img.shields.io/badge/Platform-Android%20%7C%20iOS%20%7C%20PC-blue.svg)
![Genre](https://img.shields.io/badge/Genre-Puzzle%20Game-green.svg)

**"Is it SAFE BomB"**는 폭탄을 안전 지역으로 옮기는 스트레스 해소형 퍼즐 게임입니다. 직관적인 드래그 앤 드롭 조작과 다양한 퍼즐 요소로 플레이어에게 도전과 재미를 동시에 제공합니다.

## 🎯 게임 개요

플레이어는 위험한 폭탄을 클릭하여 선택하고, 드래그하여 안전 지역(SAFE ZONE)으로 이동시켜야 합니다. 24개의精心 설계된 스테이지를 통해 점진적으로 난이도가 증가하며, 포탈, 벽 장애물 등 다양한 게임 요소가 퍼즐의 깊이를 더합니다.

## ✨ 주요 기능

### 🎮 핵심 게임플레이
- **직관적인 조작**: 마우스/터치 드래그 앤 드롭 방식의 폭탄 이동
- **스마트 충돌 감지**: 벽과 장애물에 대한 정밀한 충돌 처리
- **안전 지역 시스템**: 폭탄이 SAFE ZONE에 들어가면 카운트 감소

### 📱 모바일 최적화
- **반응형 화면 비율**: 9:16 비율 기반의 모바일 화면 최적화
- **터치 컨트롤**: 모바일 기기에서의 완벽한 터치 조작 지원
- **성능 최적화**: 경량화된 스크립트로 빠른 로딩 속도

### 🏆 스테이지 시스템
- **24개 스테이지**: 2개 챕터, 각 12개 스테이지로 구성
- **진행률 저장**: PlayerPrefs를 이용한 스테이지 클리어 데이터 자동 저장
- **스테이지 잠금/해제**: 순차적 스테이지 진행 방식

### 🎵 사용자 경험
- **BGM 제어**: 토글 버튼을 통한 배경음악 ON/OFF 기능
- **일시정지 시스템**: 게임 중간에 일시정지 및 재개 기능
- **다이내믹 배경**: 챕터별로 다른 배경 이미지 제공

## 🎨 게임 요소

### 🎯 기본 오브젝트
- **폭탄 (Bomb)**: 플레이어가 조작하는 메인 오브젝트
- **안전 지역 (SAFE)**: 폭탄이 도달해야 하는 목표 지점
- **벽 (WALL)**: 폭탄 이동을 방해하는 장애물
- **포탈 (PORTAL)**: 특정 위치로 폭탄을 순간이동시키는 장치

### 📊 스테이지 난이도 설계
- **초급 (1-8단계)**: 기본 개념 학습, 간단한 퍼즐
- **중급 (9-16단계)**: 포탈과 벽 조합, 복합적 사고 요구
- **고급 (17-24단계)**: 복잡한 레이아웃, 최적 경로 찾기

## 📱 플레이 방법

1. **폭탄 선택**: 화면의 폭탄을 클릭하여 선택
2. **드래그 이동**: 원하는 방향으로 드래그하여 폭탄 이동
3. **안전 지역 도달**: 모든 폭탄을 SAFE ZONE으로 이동시켜 스테이지 클리어
4. **다음 스테이지**: 클리어한 스테이지를 저장하고 다음 단계로 진행
